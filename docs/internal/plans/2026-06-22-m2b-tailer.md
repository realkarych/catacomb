# M2b — JSONL transcript tailer + subagent tree + JSONL precedence tier Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add Catacomb's fourth and final live ingest source — a pure-Go polling tailer over Claude Code's `~/.claude/projects/**/<session>.jsonl` transcripts — feeding newly-appended lines through the existing `ingest/jsonl` parser with a durable byte-offset cursor, activate the JSONL tier in the reducer's per-field precedence, and emit the subagent parent→child tree (the #53954 backfill).

**Architecture:** The offline `ingest/jsonl.Parse` is generalized with a `nextSeq func() uint64` seam plus an `observedAt func(time.Time) time.Time` resolver so the live tailer supplies the daemon's global gap-free `seq` and ingest-time `ObservedAt` while offline `replay` stays byte-identical via a thin `ParseReader` wrapper. A new `ingest/tail` package runs a clock-injected poll loop (a `time.Ticker` bound to `Serve`'s ctx, never `time.Sleep`; tests drive `pollOnce` directly), tracks per-file byte offsets in a new `tail_cursors` SQLite table, and detects truncation/rotation portably via (size, mtime, a first-N-bytes content fingerprint) — never unix-only `syscall.Stat_t` inode/dev. Each complete new line is parsed and routed through a new `Daemon.IngestTranscript` under `d.mu` (single-writer discipline); the reducer's `(rank, seq)` stamp mechanism gains the JSONL tier in `sourceRank`/`payloadRank`/`structureRank`, and the JSONL parser becomes a producer of `parent_tool_use_id` edges and subagent nodes consumed by the existing `upsertParentToolEdge`/`applySubagent` paths.

**Tech Stack:** Go 1.26 pure-Go; stdlib only (`os`, `bufio`, `bytes`, `crypto/sha256`, `path/filepath`, `time`, `io`, `database/sql`); `modernc.org/sqlite` (already vendored); `github.com/oklog/ulid/v2` (already vendored); `github.com/spf13/cobra` (already vendored); `github.com/stretchr/testify` for tests. NO new dependencies.

## Global Constraints

- **Go 1.26, pure-Go, no cgo, no unix-only syscalls.** `modernc.org/sqlite` only; single static cross-platform binary. **`GOOS=windows go build ./...` MUST pass** — the tailer's rotation/truncation detection uses ONLY `os.Stat` / `os.FileInfo` / `os.Open`+read; **NEVER `syscall.Stat_t` inode/dev** (unix-only, breaks the Windows build).
- **NO new dependencies.** M2b needs none (pure-Go poll via stdlib `os`/`time`/`bufio`/`crypto/sha256`). **NEVER `go mod tidy`**; if a dep were ever needed it is `go get <pkg>@latest` one at a time (it is not needed here).
- **NO comments in Go code** (`internal/codepolicy`). The only allowed comments are the `//go:build`, `//go:embed`, `//go:generate` directives; `// Code generated … DO NOT EDIT.` files are skipped wholesale.
- **100% line coverage under `-race`** (`make cover`), TDD-first, threshold never lowered. Every branch — rotation, truncation, partial-line buffering, new-file discovery, self-exclusion, lossy flagging, each JSONL rank arm — must be reached by a GENUINE assertion-bearing test.
- **NO dead code.** JSONL is now a LIVE source, so the `SourceJSONL` arms added to `sourceRank` / `payloadRank` / `structureRank` are reachable and each needs a genuine test. Do NOT add unreachable arms.
- **`golangci-lint v2` clean** (gofumpt, goimports local-prefix `github.com/realkarych/catacomb`, govet shadow, **forbidigo bans `time.Sleep`**, unparam, errcheck, rowserrcheck, bodyclose). The tailer poll loop uses a `time.Ticker` bound to ctx + a synchronous `pollOnce` test seam — **NEVER `time.Sleep`**; tests use the seam or `require.Eventually`. **unparam gotcha:** vary args/sources/paths/offsets across call sites so a parameter is never seen as constant.
- **Single-mutex daemon discipline.** `d.graphs` / `d.execBySession` / `d.lastSeen` / bus / metrics under `d.mu`. The tailer does ALL file I/O (stat, open, read, glob) OUTSIDE the lock; only the per-line ingest call (`IngestTranscript`) takes `d.mu`.
- **Observation log is the system of record** (ADR-0002); the reducer is a deterministic **COMMUTATIVE** projection (original spec §16). The `tail_cursors` table is DERIVED state (a tailing cache), not a source of truth — a rebuild re-reduces from observations. The precedence extension stays a pure function of `(rank, seq)`.
- **`seq` is the global gap-free total order** (ADR-0010/0018), stamped once at the daemon's serialized append boundary (`daemon.go` `next()`). The live tailer path takes `seq` from `d.next` via the `nextSeq` seam — never a parser-local counter. The offline `replay` path keeps its parser-local counter (byte-identical golden output).
- **`nowFn` injection** (ADR-0018). No `time.Now()` in non-test code; `ingest/jsonl` and `ingest/tail` use the `var nowFn = time.Now` seam.
- **ADR-0019 fault isolation + loop-break.** The tailer goroutine runs under `recover()` + restart; one bad line → a quarantined poison observation + loop restart, never a daemon crash. The tailer SELF-EXCLUDES Catacomb's own files (its db path + the daemon cwd transcript dir) so dogfooding cannot feed a runaway loop. On any detected gap the affected run is flagged `lossy`.
- **Cross-platform tests.** Close every opened file before `t.TempDir` cleanup (Windows holds file locks). All test files are white-box (same package).
- **markdownlint** (CI, `npx markdownlint-cli@0.49.0`, config `.markdownlint.json`): blank lines around every list and fenced code block (MD031/MD032). Run `npx markdownlint-cli@0.49.0 --fix` on this plan before committing.

```go
func ParseReader(r io.Reader, executionID string) ([]model.Observation, error) {
	var seq uint64
	next := func() uint64 {
		s := seq
		seq++
		return s
	}
	return Parse(r, executionID, next, func(eventTime time.Time) time.Time { return eventTime })
}

func Parse(r io.Reader, executionID string, nextSeq func() uint64, observedAt func(eventTime time.Time) time.Time) ([]model.Observation, error) {
	sc := bufio.NewScanner(r)
	sc.Buffer(make([]byte, 0, 1024*1024), 16*1024*1024)

	var out []model.Observation
	for sc.Scan() {
		raw := strings.TrimSpace(sc.Text())
		if raw == "" {
			continue
		}
		ln, parts, err := decodeLine([]byte(raw))
		if err != nil {
			return nil, err
		}
		ts, _ := time.Parse(time.RFC3339, ln.Timestamp)
		for _, p := range parts {
			out = append(out, model.Observation{
				ObsID:       ulid.Make().String(),
				RunID:       ln.SessionID,
				ExecutionID: executionID,
				Source:      model.SourceJSONL,
				Kind:        p.kind,
				Correlation: p.correlation,
				Attrs:       p.attrs,
				Payload:     p.payload,
				EventTime:   ts,
				ObservedAt:  observedAt(ts),
				Seq:         nextSeq(),
			})
		}
	}
	if err := sc.Err(); err != nil {
		return nil, fmt.Errorf("jsonl.ParseReader: %w", err)
	}
	return out, nil
}
```

### Task 5: `ingest/tail` poll tailer (cursor, partial-line, portable rotation/truncation, discovery, self-exclusion, lossy)

**Files:**

- Create: `ingest/tail/tail.go`
- Test: `ingest/tail/tail_test.go`

**Interfaces:**

- Consumes: `model.TailCursor`; `ingest/jsonl.Parse` (Task 1 signature). Defines a small sink interface so the daemon (Task 6) plugs in without an import cycle.
- Produces:
  - `type Sink interface { IngestTranscript(line []byte, sessionID string) error; MarkLossy(sessionID string) }` — the tailer's only outward dependency (Task 6's `Daemon` satisfies it; the method is named `IngestTranscript` because `Daemon.Ingest(hookType, payload)` already exists with a different signature).
  - `type Store interface { LoadTailCursors() ([]model.TailCursor, error); UpsertTailCursor(c model.TailCursor) error }` — satisfied by `store.Store` (structural subset).
  - `type Tailer struct { … }` with `func New(dir string, excludes []string, st Store, sink Sink) *Tailer`.
  - `func (tl *Tailer) Load() error` — seeds in-memory cursors from `st.LoadTailCursors()`.
  - `func (tl *Tailer) PollOnce(ctx context.Context) error` — one synchronous poll (the deterministic test seam): re-glob (discovery), then per non-excluded file detect rotation/truncation, read `[offset,size)`, buffer the trailing partial line, parse + sink each complete line, persist the advanced cursor. Returns the first hard error (a parse/sink error does NOT abort the poll — it marks lossy + quarantines via the sink and continues).
  - `func (tl *Tailer) Run(ctx context.Context, tick time.Duration)` — production loop: `ticker := time.NewTicker(tick); defer ticker.Stop(); for { select { case <-ctx.Done(): return; case <-ticker.C: _ = tl.PollOnce(ctx) } }`. NEVER `time.Sleep`.
  - `var nowFn = time.Now`; fingerprint = `func fingerprint(b []byte) string` = hex sha256 of up to the first `fpBytes = 512` bytes.

**Portable rotation/truncation design (the #1 risk — NO unix syscalls):** per file the tailer stores `model.TailCursor{Path, Offset, Fingerprint, Size, Mtime}`. On each poll it `os.Stat`s the path (`FileInfo.Size()`, `ModTime().UnixNano()`), then:

- **Truncation:** `info.Size() < cursor.Offset` → the file shrank → reset `Offset = 0`, recompute the fingerprint, mark the session lossy, re-read from the start.
- **Rotation (same path, file replaced):** read the current first `fpBytes` and compare `fingerprint(head)` against `cursor.Fingerprint`. If the stored offset is non-zero and the new head fingerprint differs from the stored one (the bytes that were at the file head changed) → the file was replaced under the same path → reset `Offset = 0`, mark lossy, re-read. This is the cross-platform replacement for unix inode/dev: an inode change manifests as a different head-byte fingerprint. (`Size`/`Mtime` are stored for cheap pre-checks and operator debugging; the fingerprint is the authoritative rotation signal.)
- **New file at a new path:** the periodic re-glob discovers it with no stored cursor → a fresh `Offset = 0` (the path is the key).
- **Append (normal):** `info.Size() > cursor.Offset` and head fingerprint matches → read `[Offset, Size)`.

ALL of this uses only `os.Stat` / `os.Open` / `(*os.File).ReadAt` / `io.ReadFull` — portable; `GOOS=windows go build ./...` stays clean.

**Partial-line handling:** read the new bytes into a buffer prepended with the per-file in-memory `pending` remainder. Split on `\n`; the bytes after the last `\n` are an incomplete line → keep them in `pending`, advance `Offset` ONLY by the count of fully-consumed (newline-terminated) bytes. A line spanning two polls is never half-parsed.

**Self-exclusion (ADR-0019 loop-break):** `New` stores the daemon db path + the daemon cwd-encoded transcript subdir in `excludes`; `PollOnce` skips any globbed path matching an exclude glob (via `filepath.Match` on the base name and a substring check on the db path) and never opens Catacomb's own db.

```go
var nowFn = time.Now

const fpBytes = 512

type Sink interface {
	IngestTranscript(line []byte, sessionID string) error
	MarkLossy(sessionID string)

type Store interface {
	LoadTailCursors() ([]model.TailCursor, error)
	UpsertTailCursor(c model.TailCursor) error
}

type fileState struct {
	cursor  model.TailCursor
	pending []byte
}

type Tailer struct {
	dir      string
	excludes []string
	store    Store
	sink     Sink
	files    map[string]*fileState
}

func New(dir string, excludes []string, st Store, sink Sink) *Tailer {
	return &Tailer{dir: dir, excludes: excludes, store: st, sink: sink, files: map[string]*fileState{}}
}

func (tl *Tailer) Load() error {
	cs, err := tl.store.LoadTailCursors()
	if err != nil {
		return fmt.Errorf("tail.Load: %w", err)
	}
	for _, c := range cs {
		tl.files[c.Path] = &fileState{cursor: c}
	}
	return nil
}

func (tl *Tailer) Run(ctx context.Context, tick time.Duration) {
	ticker := time.NewTicker(tick)
	defer ticker.Stop()
	for {
		select {
		case <-ctx.Done():
			return
		case <-ticker.C:
			_ = tl.PollOnce(ctx)
		}
	}
}

func (tl *Tailer) glob() []string {
	var out []string
	main, _ := filepath.Glob(filepath.Join(tl.dir, "*", "*.jsonl"))
	sub, _ := filepath.Glob(filepath.Join(tl.dir, "*", "*", "subagents", "agent-*.jsonl"))
	out = append(out, main...)
	out = append(out, sub...)
	return out
}

func (tl *Tailer) excluded(path string) bool {
	for _, ex := range tl.excludes {
		if ex == "" {
			continue
		}
		if strings.Contains(path, ex) {
			return true
		}
		if ok, _ := filepath.Match(ex, filepath.Base(path)); ok {
			return true
		}
	}
	return false
}

func sessionOf(path string) string {
	return strings.TrimSuffix(filepath.Base(path), ".jsonl")
}

func fingerprint(b []byte) string {
	if len(b) > fpBytes {
		b = b[:fpBytes]
	}
	sum := sha256.Sum256(b)
	return hex.EncodeToString(sum[:])
}

func headFingerprint(path string) (string, error) {
	f, err := os.Open(path)
	if err != nil {
		return "", err
	}
	defer func() { _ = f.Close() }()
	buf := make([]byte, fpBytes)
	n, err := io.ReadFull(f, buf)
	if err != nil && err != io.EOF && err != io.ErrUnexpectedEOF {
		return "", err
	}
	return fingerprint(buf[:n]), nil
}

func (tl *Tailer) PollOnce(ctx context.Context) error {
	for _, path := range tl.glob() {
		if err := ctx.Err(); err != nil {
			return err
		}
		if tl.excluded(path) {
			continue
		}
		if err := tl.pollFile(path); err != nil {
			return err
		}
	}
	return nil
}

func (tl *Tailer) pollFile(path string) error {
	info, err := os.Stat(path)
	if err != nil {
		return nil
	}
	st, ok := tl.files[path]
	if !ok {
		st = &fileState{cursor: model.TailCursor{Path: path}}
		tl.files[path] = st
	}
	size := info.Size()
	session := sessionOf(path)

	head, err := headFingerprint(path)
	if err != nil {
		return nil
	}
	if size < st.cursor.Offset {
		st.cursor.Offset = 0
		st.pending = nil
		tl.sink.MarkLossy(session)
	} else if st.cursor.Offset > 0 && st.cursor.Fingerprint != "" && head != st.cursor.Fingerprint {
		st.cursor.Offset = 0
		st.pending = nil
		tl.sink.MarkLossy(session)
	}
	if size == st.cursor.Offset {
		return tl.persist(st, head, info)
	}

	f, err := os.Open(path)
	if err != nil {
		return nil
	}
	defer func() { _ = f.Close() }()
	buf := make([]byte, size-st.cursor.Offset)
	n, err := f.ReadAt(buf, st.cursor.Offset)
	if err != nil && err != io.EOF {
		return nil
	}
	data := append(st.pending, buf[:n]...)
	advance := bytes.LastIndexByte(data, '\n')
	if advance < 0 {
		st.pending = data
		return tl.persist(st, head, info)
	}
	complete := data[:advance+1]
	st.pending = append([]byte{}, data[advance+1:]...)
	for _, raw := range bytes.Split(complete, []byte{'\n'}) {
		trimmed := bytes.TrimSpace(raw)
		if len(trimmed) == 0 {
			continue
		}
		if err := tl.sink.IngestTranscript(trimmed, session); err != nil {
			tl.sink.MarkLossy(session)
		}
	}
	st.cursor.Offset += int64(advance + 1)
	return tl.persist(st, head, info)
}

func (tl *Tailer) persist(st *fileState, head string, info os.FileInfo) error {
	st.cursor.Fingerprint = head
	st.cursor.Size = info.Size()
	st.cursor.Mtime = info.ModTime().UnixNano()
	if err := tl.store.UpsertTailCursor(st.cursor); err != nil {
		return fmt.Errorf("tail.persist: %w", err)
	}
	return nil
}
```

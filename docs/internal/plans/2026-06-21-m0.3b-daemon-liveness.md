# M0.3b — Daemon liveness & resilience Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Make the live daemon resilient and self-finalizing — one malformed hook never crashes it (per-request `recover()` + a poison-quarantine table), the runs projection is persisted live and rebuilt on boot, and idle runs are reaped to `abandoned` via a durable synthetic `run_ended` observation.

**Architecture:** All daemon-side, building on the merged M0.3a reducer/store. `Ingest` is wrapped in a `recover()` boundary that quarantines bad input and stays fail-open; a shared `applyAndPersist` helper folds + persists nodes/edges + the run projection; an idle-reaper ticker bound to the serve context appends a synthetic `run_ended{reason:timeout}` for quiescent runs so the reducer marks them `abandoned` and closes their open descendants. Time is taken through a `nowFn` seam; the fault path through an `applyFn` seam.

**Tech Stack:** Go 1.26, pure-Go `modernc.org/sqlite` (no cgo), `oklog/ulid/v2`, `spf13/cobra`, `testify`.

## Global Constraints

- Go 1.26; pure-Go only (`modernc.org/sqlite`, no cgo).
- **No comments in Go source** except `//go:build`, `//go:embed`, `//go:generate` (enforced by `internal/codepolicy`). Not even doc comments.
- **100% line coverage** under `-race`, TDD-first (`make cover`). The threshold never lowers.
- `golangci-lint v2` clean (`make lint`): gofumpt, goimports local-prefix `github.com/realkarych/catacomb`, govet shadow, **forbidigo bans `time.Sleep`**, unparam, errcheck, rowserrcheck, bodyclose.
- **Never run `go mod tidy`.** If a dependency seems missing, stop and report.
- The **observation log is the system of record**; the `runs` and `quarantine` tables are derived/diagnostic. Run state rebuilds from the log via `ObservationsSince(0)` + `Apply`.
- Cross-platform (Linux/macOS/Windows): **no `time.Sleep`** (use `time.Ticker`/`require.Eventually`), no unix-socket assumptions, no `0600`-mode assertions, time only via the `nowFn` seam; close the store/listener before any `t.TempDir()` cleanup (Windows file-lock).

```go
func (d *Daemon) Ingest(hookType string, payload []byte) (err error) {
	d.mu.Lock()
	defer d.mu.Unlock()
	defer func() {
		if r := recover(); r != nil {
			d.quarantine(hookType, payload, fmt.Sprintf("panic: %v", r))
			err = nil
		}
	}()
	if e := d.ingestLocked(hookType, payload); e != nil {
		d.quarantine(hookType, payload, e.Error())
	}
	return nil
}

func (d *Daemon) ingestLocked(hookType string, payload []byte) error {
	sessionID := sessionIDOf(payload)
	execID, ok := d.execBySession[sessionID]
	if !ok {
		execID = d.newExecID()
		d.execBySession[sessionID] = execID
	}
	obs, err := hook.Parse(hookType, payload, execID, d.next)
	if err != nil {
		return err
	}
	g, ok := d.graphs[execID]
	if !ok {
		g = reduce.NewGraph()
		d.graphs[execID] = g
	}
	for _, o := range obs {
		applyFn(g, o)
		nodes, edges := g.Snapshot()
		if err := d.store.AppendAndApply(o, nodes, edges); err != nil {
			return err
		}
	}
	return nil
}

func (d *Daemon) quarantine(hookType string, payload []byte, msg string) {
	d.quarantined++
	rec := model.QuarantineRecord{Raw: payload, HookType: hookType, Err: msg, At: nowFn().UTC()}
	if err := d.store.Quarantine(rec); err != nil {
		log.Printf("catacomb: quarantine write failed: %v", err)
	}
}
```

Add `"fmt"`, `"log"`, `"time"` to the imports (drop any now-unused). `model` is already imported.

```go
func (d *Daemon) reapIdle(now time.Time) error {
	d.mu.Lock()
	defer d.mu.Unlock()
	for execID, g := range d.graphs {
		for runID, r := range g.Runs {
			if r.Status != model.StatusRunning {
				continue
			}
			last := d.lastSeen[runID]
			if now.Sub(last) < d.reaperWindow {
				continue
			}
			o := model.Observation{
				ObsID:       ulid.Make().String(),
				RunID:       runID,
				ExecutionID: execID,
				Source:      model.SourceHook,
				Kind:        "run_ended",
				Attrs:       map[string]any{"reason": "timeout"},
				EventTime:   now.UTC(),
				ObservedAt:  now.UTC(),
				Seq:         d.next(),
			}
			if err := d.applyAndPersist(g, o); err != nil {
				return err
			}
		}
	}
	return nil
}
```

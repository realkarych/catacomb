# Catacomb M0.2b — Daemon & CLI — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Make Catacomb go live end-to-end: a long-running daemon receives Claude Code hook events over a `0600` unix-domain socket, builds the execution graph in real time on the M0.2a core, persists each observation atomically, and recovers on restart — driven by a `catacomb hook` forwarder and wired in by `catacomb install-hooks`.

**Architecture:** A new `daemon` package owns the socket, the in-memory `reduce.Graph` per execution, the live ingest pipeline (hook payload → `ingest/hook.Parse` → `reduce.Apply` → `store.AppendAndApply`), and boot recovery (replay `store.ObservationsSince(0)` through the reducer). It serves HTTP/1.1 over the unix socket (`POST /hook/{type}`, `GET /healthz`). Three thin `cmd/catacomb` subcommands wrap it: `daemon` (run it), `hook <type>` (a fail-open forwarder that POSTs the hook's stdin to the socket), `install-hooks` (idempotent `settings.json` wiring). Ingestion is serialized under one mutex — correct and simple at interactive rates.

**Tech Stack:** Go 1.26 (uses 1.22+ `net/http` method+wildcard routing and `Request.PathValue`), `modernc.org/sqlite`, `github.com/oklog/ulid/v2`, `github.com/spf13/cobra`, `github.com/stretchr/testify`. No new dependencies.

> **Transport primitives pre-validated.** A throwaway scratch module confirmed (race-clean): unix-socket `http.Server` + a unix-dialing `http.Client` round-trip; `POST /hook/{type}` + `r.PathValue("type")` routing; the socket carries `0600` after `os.Chmod`; a client to an absent socket returns a dial error (the forwarder's fail-open trigger); and `net.Listen("unix", p)` fails with "address already in use" if `p` exists (so the daemon must `os.Remove` a stale socket first). The plan's transport code uses exactly these shapes. **Not otherwise pre-reconstructed** — the per-task TDD loop (RED→GREEN, `make cover` runs `-race`) plus the review gates are the validator. The hook payload field names and the `settings.json` hooks shape are the undocumented surfaces; Task 2 reuses M0.2a's `ingest/hook` fixtures as the contract, and Task 6 includes a one-time real-session verification.

```go
package daemon

import (
	"fmt"
	"net"
	"os"
	"path/filepath"
)

var (
	netListen = net.Listen
	osChmod   = os.Chmod
)

func SocketPath() string {
	if p := os.Getenv("CATACOMB_SOCKET"); p != "" {
		return p
	}
	if dir := os.Getenv("XDG_RUNTIME_DIR"); dir != "" {
		return filepath.Join(dir, "catacomb", "daemon.sock")
	}
	home, err := os.UserHomeDir()
	if err != nil {
		return filepath.Join(os.TempDir(), "catacomb", "daemon.sock")
	}
	return filepath.Join(home, ".catacomb", "run", "daemon.sock")
}

func Listen(path string) (net.Listener, error) {
	if err := os.MkdirAll(filepath.Dir(path), 0o700); err != nil {
		return nil, fmt.Errorf("daemon.Listen mkdir: %w", err)
	}
	if err := os.Remove(path); err != nil && !os.IsNotExist(err) {
		return nil, fmt.Errorf("daemon.Listen stale: %w", err)
	}
	ln, err := netListen("unix", path)
	if err != nil {
		return nil, fmt.Errorf("daemon.Listen: %w", err)
	}
	if err := osChmod(path, 0o600); err != nil {
		_ = ln.Close()
		return nil, fmt.Errorf("daemon.Listen chmod: %w", err)
	}
	return ln, nil
}
```

### Task 2: `daemon/daemon.go` — live ingest core + recovery (no HTTP yet)

The `Daemon` holds the store, a per-execution in-memory `reduce.Graph`, a session→execution map, and the global `seq`, all under one mutex. `Ingest` resolves the execution id for the event's session (minting one for a new session), parses the hook payload, applies each observation to the graph, and persists it with the current graph snapshot in one transaction. `Recover` replays the durable observation log to rebuild the in-memory graphs, the session map, and `seq` after a restart. Per-event whole-graph snapshot persistence is O(graph) per event — fine at interactive rates (YAGNI to optimize; revisit only if measured).

```go
package daemon

import (
	"encoding/json"
	"sync"

	"github.com/oklog/ulid/v2"

	"github.com/realkarych/catacomb/ingest/hook"
	"github.com/realkarych/catacomb/reduce"
	"github.com/realkarych/catacomb/store"
)

type Daemon struct {
	store         store.Store
	newExecID     func() string
	mu            sync.Mutex
	seq           uint64
	graphs        map[string]*reduce.Graph
	execBySession map[string]string
}

func New(s store.Store) *Daemon {
	return &Daemon{
		store:         s,
		newExecID:     func() string { return ulid.Make().String() },
		graphs:        map[string]*reduce.Graph{},
		execBySession: map[string]string{},
	}
}

func (d *Daemon) Recover() error {
	obs, err := d.store.ObservationsSince(0)
	if err != nil {
		return err
	}
	var max uint64
	for _, o := range obs {
		g, ok := d.graphs[o.ExecutionID]
		if !ok {
			g = reduce.NewGraph()
			d.graphs[o.ExecutionID] = g
		}
		g.Apply(o)
		d.execBySession[o.RunID] = o.ExecutionID
		if o.Seq > max {
			max = o.Seq
		}
	}
	d.seq = max
	return nil
}

func (d *Daemon) Ingest(hookType string, payload []byte) error {
	d.mu.Lock()
	defer d.mu.Unlock()

	sessionID := sessionIDOf(payload)
	execID, ok := d.execBySession[sessionID]
	if !ok {
		execID = d.newExecID()
		d.execBySession[sessionID] = execID
	}
	obs, err := hook.Parse(hookType, payload, execID, d.next)
	if err != nil {
		return err
	}
	g, ok := d.graphs[execID]
	if !ok {
		g = reduce.NewGraph()
		d.graphs[execID] = g
	}
	for _, o := range obs {
		g.Apply(o)
		nodes, edges := g.Snapshot()
		if err := d.store.AppendAndApply(o, nodes, edges); err != nil {
			return err
		}
	}
	return nil
}

func (d *Daemon) next() uint64 {
	d.seq++
	return d.seq
}

func sessionIDOf(payload []byte) string {
	var e struct {
		SessionID string `json:"session_id"`
	}
	if err := json.Unmarshal(payload, &e); err != nil {
		return ""
	}
	return e.SessionID
}
```

> `next` is the method value passed as `nextSeq` to `hook.Parse`; it runs under `d.mu` because only `Ingest` calls it. `Recover` runs once at boot before `Serve`, so it needs no lock.

```go
package daemon

import (
	"context"
	"errors"
	"io"
	"net"
	"net/http"
)

func (d *Daemon) Handler() http.Handler {
	mux := http.NewServeMux()
	mux.HandleFunc("POST /hook/{type}", d.handleHook)
	mux.HandleFunc("GET /healthz", func(w http.ResponseWriter, _ *http.Request) {
		w.WriteHeader(http.StatusOK)
	})
	return mux
}

func (d *Daemon) handleHook(w http.ResponseWriter, r *http.Request) {
	payload, err := io.ReadAll(r.Body)
	if err != nil {
		w.WriteHeader(http.StatusBadRequest)
		return
	}
	if err := d.Ingest(r.PathValue("type"), payload); err != nil {
		w.WriteHeader(http.StatusInternalServerError)
		return
	}
	w.WriteHeader(http.StatusNoContent)
}

func (d *Daemon) Serve(ctx context.Context, ln net.Listener) error {
	srv := &http.Server{Handler: d.Handler()}
	ctx, cancel := context.WithCancel(ctx)
	defer cancel()
	go func() {
		<-ctx.Done()
		_ = srv.Close()
	}()
	if err := srv.Serve(ln); err != nil && !errors.Is(err, http.ErrServerClosed) {
		return err
	}
	return nil
}
```

> `Serve` blocks until `ctx` is cancelled (graceful shutdown — the `daemon` command wires `signal.NotifyContext` so Ctrl-C/SIGTERM stops it) or the listener fails. The internal `context.WithCancel`+`defer cancel()` guarantees the shutdown goroutine always exits when `Serve` returns (no leak on either path), and `srv.Close()` closes the listener so the caller's deferred `s.Close()` runs — important so tests do not leave the SQLite handle open (Windows `t.TempDir()` cleanup fails on an open file). The DialContext param is named `c` to avoid shadowing the outer `ctx` (govet shadow).

```go
import (
	"encoding/json"
	"fmt"
	"os"
	"path/filepath"
	"strings"

	"github.com/spf13/cobra"

	"github.com/realkarych/catacomb/daemon"
)

var hookEvents = []string{
	"SessionStart", "UserPromptSubmit", "PreToolUse", "PostToolUse",
	"SubagentStop", "Stop", "SessionEnd", "PreCompact", "Notification",
}

var osExecutable = os.Executable

func newInstallHooksCmd() *cobra.Command {
	var project, global, uninstall bool
	cmd := &cobra.Command{
		Use:   "install-hooks",
		Short: "Wire the catacomb hook forwarder into Claude Code settings.json",
		RunE: func(cmd *cobra.Command, _ []string) error {
			path, err := settingsPath(project, global)
			if err != nil {
				return err
			}
			exe, err := osExecutable()
			if err != nil {
				return fmt.Errorf("install-hooks executable: %w", err)
			}
			return installHooks(path, daemon.SocketPath(), exe, uninstall)
		},
	}
	cmd.Flags().BoolVar(&project, "project", false, "write .claude/settings.json in the current directory (default)")
	cmd.Flags().BoolVar(&global, "global", false, "write ~/.claude/settings.json")
	cmd.Flags().BoolVar(&uninstall, "uninstall", false, "remove catacomb hook entries")
	return cmd
}

func settingsPath(project, global bool) (string, error) {
	if project && global {
		return "", fmt.Errorf("install-hooks: choose at most one of --project / --global")
	}
	if global {
		home, err := os.UserHomeDir()
		if err != nil {
			return "", fmt.Errorf("install-hooks home: %w", err)
		}
		return filepath.Join(home, ".claude", "settings.json"), nil
	}
	return filepath.Join(".claude", "settings.json"), nil
}

func installHooks(path, socket, exe string, uninstall bool) error {
	settings, err := readSettings(path)
	if err != nil {
		return err
	}
	hooks := asMap(settings["hooks"])
	marker := exe + " hook "
	for _, ev := range hookEvents {
		entries := pruneCatacomb(asSlice(hooks[ev]), marker)
		if !uninstall {
			entries = append(entries, catacombEntry(socket, exe, ev))
		}
		if len(entries) == 0 {
			delete(hooks, ev)
		} else {
			hooks[ev] = entries
		}
	}
	if len(hooks) == 0 {
		delete(settings, "hooks")
	} else {
		settings["hooks"] = hooks
	}
	return writeSettings(path, settings)
}

func readSettings(path string) (map[string]any, error) {
	b, err := os.ReadFile(path)
	if err != nil {
		if os.IsNotExist(err) {
			return map[string]any{}, nil
		}
		return nil, fmt.Errorf("install-hooks read: %w", err)
	}
	var settings map[string]any
	if err := json.Unmarshal(b, &settings); err != nil {
		return nil, fmt.Errorf("install-hooks parse: %w", err)
	}
	if settings == nil {
		settings = map[string]any{}
	}
	return settings, nil
}

func writeSettings(path string, settings map[string]any) error {
	if err := os.MkdirAll(filepath.Dir(path), 0o755); err != nil {
		return fmt.Errorf("install-hooks mkdir: %w", err)
	}
	b, err := json.MarshalIndent(settings, "", "  ")
	if err != nil {
		return fmt.Errorf("install-hooks marshal: %w", err)
	}
	if err := os.WriteFile(path, append(b, '\n'), 0o644); err != nil {
		return fmt.Errorf("install-hooks write: %w", err)
	}
	return nil
}

func catacombEntry(socket, exe, event string) map[string]any {
	return map[string]any{
		"matcher": "",
		"hooks": []any{map[string]any{
			"type":    "command",
			"command": fmt.Sprintf("CATACOMB_SOCKET=%s %s hook %s", socket, exe, event),
		}},
	}
}

func pruneCatacomb(entries []any, marker string) []any {
	out := make([]any, 0, len(entries))
	for _, e := range entries {
		if !entryHasMarker(e, marker) {
			out = append(out, e)
		}
	}
	return out
}

func entryHasMarker(entry any, marker string) bool {
	m, ok := entry.(map[string]any)
	if !ok {
		return false
	}
	for _, h := range asSlice(m["hooks"]) {
		hm, ok := h.(map[string]any)
		if !ok {
			continue
		}
		if cmd, ok := hm["command"].(string); ok && strings.Contains(cmd, marker) {
			return true
		}
	}
```

```go
package main

import (
	"context"
	"net"
	"os"
	"os/signal"
	"syscall"

	"github.com/spf13/cobra"

	"github.com/realkarych/catacomb/daemon"
	"github.com/realkarych/catacomb/store"
)

func newDaemonCmd() *cobra.Command {
	var dbPath, socketPath string
	cmd := &cobra.Command{
		Use:   "daemon",
		Short: "Run the catacomb daemon (receives hook events, builds the live graph)",
		RunE: func(cmd *cobra.Command, _ []string) error {
			if socketPath == "" {
				socketPath = daemon.SocketPath()
			}
			ctx, stop := signal.NotifyContext(cmd.Context(), os.Interrupt, syscall.SIGTERM)
			defer stop()
			return runDaemonWith(ctx, store.OpenSQLite, daemon.Listen, dbPath, socketPath)
		},
	}
	cmd.Flags().StringVar(&dbPath, "db", "catacomb.db", "SQLite database path")
	cmd.Flags().StringVar(&socketPath, "socket", "", "unix socket path (default: resolved CATACOMB_SOCKET)")
	return cmd
}

func runDaemonWith(ctx context.Context, open func(string) (store.Store, error), listen func(string) (net.Listener, error), dbPath, socketPath string) error {
	s, err := open(dbPath)
	if err != nil {
		return err
	}
	defer func() { _ = s.Close() }()

	d := daemon.New(s)
	if err := d.Recover(); err != nil {
		return err
	}
	ln, err := listen(socketPath)
	if err != nil {
		return err
	}
	defer func() { _ = ln.Close() }()

	return d.Serve(ctx, ln)
}
```

> The `daemon` command derives `ctx` from `signal.NotifyContext` so Ctrl-C/SIGTERM triggers a graceful shutdown and `runDaemonWith` returns (closing the store and listener via its defers). Tests pass a cancelable context and `cancel()` to stop the daemon deterministically — never leaving the SQLite handle open at `t.TempDir()` cleanup.
